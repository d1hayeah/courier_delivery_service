# Courier Delivery Service

## Описание

**Courier Delivery Service** - это программа для службы курьерской доставки. Программа позволяет работать с курьерами, заказами и отслеживать заказы по трекинг номеру. 

## Цель проекта
- Автоматизировать учет и управление службой доставки

## Структура
- **auth/** - модули для авторизации/регистрации и смены пароля
- **database/** - модуль подключения к базе данных
- **modules/** - модули для окон приложения
- **utils/** - модуль для формирования отчётов
- **main.py** - файл для запуска проекта

## Как установить и запустить?
- Нажать в репозитории на Code и после на download zip 
- Распаковать ZIP-файл проекта
- Указать в консоли путь в папку: cd (путь)
- Запустить проект через команду: python main.py

# Важно
Нужно иметь установленный python и библиотеки PyQt5,hashlib и psycopg2
